# Student management
A Full stack MERN website for students <br/>
<small>This project will be completed and hosted very soon! </small>

<h2>Installation </h2>

Use the package manager [npm] to install.
Setup the project and install the packages by running: 

```bash
npm run setup
```
 Run project with command
```bash
npm run dev
```
 
<h2> Built with  </h2>
<ul>
  <li>FrontEnd: <b> React.JS, material-ui, HTML/CSS </b></li>
  <li>Backend:  <b> Node.JS, Express.JS </b> </li>
  <li>Database: <b> MongoDB, Mongoose </b> </li>
</ul>

<h2> Features </h2>
<ul>
  <li> Add a new student to the list.</li>
  <li> Delete student from the list.</li>
</ul>

<h2> REST API </h2>
<h4> Students </h4>
<ul>
  <li> <b>GET</b>  /api/students </li>
  <li> <b>POST</b>  /api/students,student  </li>
  <li> <b>DELETE</b>  /api//students/:_id </li>
</ul>
